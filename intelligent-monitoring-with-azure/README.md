## Azure Monitoring Tools Overview

### Azure Advisor

Azure Advisor is a personalized cloud consultant that helps you follow best practices to optimize your Azure deployments.

- Analyzes your deployed resources and configurations.
- Provides actionable recommendations across five pillars:
  - **Reliability**: Ensure and improve the continuity of your business-critical applications.
  - **Security**: Detect threats and vulnerabilities that might lead to security breaches.
  - **Performance**: Improve the speed and responsiveness of your applications.
  - **Cost**: Optimize and reduce your overall Azure spending.
  - **Operational Excellence**: Help achieve process and workflow efficiency, manageability, and best practices.

### Azure Service Health

Azure Service Health is a suite of experiences that keeps you informed about the status of your Azure services and environment.

- **Azure Status**: A broad, global view of the health of all Azure services across all regions worldwide. Good for tracking widespread issues.
- **Service Health**: A personalized dashboard providing alerts and guidance focused *only* on the specific services and regions you are currently using.
- **Resource Health**: Provides granular information about the current health of your specific, individual cloud resources (e.g., a specific virtual machine or database).

### Azure Monitor

Azure Monitor provides a comprehensive solution for collecting, analyzing, and acting on telemetry data from your cloud and on-premises environments.

- Helps maximize the availability and performance of applications and services.
- Leverages telemetry to gain insights into how applications are performing and proactively identifies issues affecting them and the resources they depend on.
- Core features encompass Log Analytics, Alerts, and Application Insights.
