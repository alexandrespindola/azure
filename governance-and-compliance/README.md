## Azure Governance and Compliance Overview

### Azure Policy

Azure Policy helps enforce organizational standards and continuously assess compliance at scale.

- Evaluates and identifies Azure resources that do not comply with organizational policies.
- Ensures resource governance and consistency across regulatory compliance, security, cost, and management.
- Provides built-in policy and initiative definitions categorized by resource types like storage, networking, compute, security center, and monitoring.

### Resource Locks

Resource Locks act as a safeguard to protect your critical Azure resources.

- Prevent accidental deletion or modification of resources.
- Can be managed and applied at the subscription, resource group, or individual resource level within the Azure Portal.

### Service Trust Portal

The Microsoft Service Trust Portal provides details about how Microsoft implements and monitors security, privacy, and compliance.

- It serves as the primary resource for accessing audit reports and compliance-related documentation regarding Microsoft's cloud services.

### Microsoft Purview

Microsoft Purview is a comprehensive family of data governance, risk, and compliance solutions.

- Provides a single, unified view of your data landscape.
- Brings together insights across on-premises, multi-cloud, and Software-as-a-Service (SaaS) data.
- **Key Features Include:**
  - Automated data discovery
  - Sensitive data classification
  - End-to-end data lineage visualization
